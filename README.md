# Employee Attendance & Salary Analytics Dashboard

## Overview

The Employee Attendance & Salary Analytics Dashboard is an interactive Power BI solution designed to help HR teams and management monitor workforce attendance and salary-related metrics efficiently. The dashboard transforms raw employee data into meaningful insights, enabling data-driven decision-making and improved workforce management.

---

## Project Objectives

- Monitor employee attendance performance.
- Analyze salary distribution across employees.
- Track workforce trends and patterns.
- Improve HR reporting efficiency.
- Provide actionable insights through visual analytics.

---

## Tools & Technologies

- Power BI Desktop
- Microsoft Excel
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization

---

## Key Features

### Attendance Analysis
- Employee attendance tracking
- Present and absent employee monitoring
- Attendance trend analysis
- Attendance percentage calculation

### Salary Analysis
- Salary distribution insights
- Average salary analysis
- Total salary expenditure tracking
- Employee salary comparison

### Interactive Dashboard
- Dynamic slicers and filters
- KPI cards for quick insights
- User-friendly visualizations
- Real-time data exploration
